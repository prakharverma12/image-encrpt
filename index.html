Security Notes
  • Passphrase strength: Use ≥ 5 random words agreed on in person — e.g. blue-elephant-dance-midnight-2026. Short passwords are brute-forceable.
  • Never hardcode it: Do not edit the HTML to include the passphrase before uploading to GitHub. The field must stay blank in the source.
  • Save this file locally: Download this HTML to your phone as a backup so it works even if the hosted URL changes.
  • Zero RAM trace: Closing or refreshing the tab permanently wipes the decrypted image — re-upload the .enc file to view it again.
  • Fully offline: Switch to Airplane Mode after the page loads; encryption and decryption work without any internet connection.
  • Crypto spec: AES-256-GCM authenticated encryption with PBKDF2-SHA256 key stretching at 310,000 iterations (OWASP 2023 minimum).
  • iOS note: On iPhone/iPad, use the hosted HTTPS GitHub Pages URL — Web Crypto is disabled on file:// URLs in Safari.
